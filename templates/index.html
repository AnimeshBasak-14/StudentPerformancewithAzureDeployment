About the Project

The Student Exam Performance Indicator is designed to analyze and predict students' academic performance based on various input factors. This project utilizes advanced machine learning techniques and is deployed on an AWS EC2 instance with containerized management via AWS Elastic Container Registry (ECR). It uses the Kaggle dataset: Students Performance in Exams.

The system predicts the math score of a student based on other factors such as gender, parental education level, test preparation, and scores in reading and writing.

Dataset Details

The dataset contains the following columns, which serve as features for prediction:

  • Gender: Indicates the gender of the student (e.g., male or female).
  • Race/Ethnicity: Categorizes students based on race/ethnicity (e.g., group A, group B, etc.).
  • Parental Level of Education: Represents the highest level of education completed by the student's parent or guardian.
  • Lunch: Indicates the type of lunch the student receives (e.g., standard or free/reduced).
  • Test Preparation Course: Specifies whether the student completed a test preparation course (completed or none).
  • Reading Score: The score obtained by the student in the reading test.
  • Writing Score: The score obtained by the student in the writing test.

Key Features

Our application provides:

  • An intuitive interface for entering student details.
  • Real-time predictions of math scores based on input data.
  • A simple and effective user experience.

Deployment Details

This project is deployed on an AWS EC2 instance, with container management via AWS Elastic Container Registry (ECR). The machine learning model was trained using the features outlined above to accurately predict math scores. Explore the source code on GitHub.
